# HideZero Gumroad — Git Architecture

## Remote
- **Repository:** https://github.com/avikla/HideZero-Gumroad
- **Default branch:** master
- **Clone:** `git clone https://github.com/avikla/HideZero-Gumroad.git`

## Backup Strategy
- **Primary:** GitHub remote (https://github.com/avikla/HideZero-Gumroad)
- **Backup location:** `../Claude-Code-Config/dotfiles/backups/HideZero-Gumroad/` (synced hourly)

## Deployment
1. Make changes locally in `.gs` files or `site/` HTML/CSS
2. Hook auto-runs `clasp push` for GAS (or run manually)
3. GitHub Pages auto-deploys `site/` → `docs/`

## Secrets Management
- **Storage:** Google Apps Script → Script Properties panel
- **Keys required:**
  - `GUMROAD_PRODUCT_ID`
  - `GUMROAD_SUB_ID`
- **Never commit:** API keys, credentials, tokens
- **How to add:** Project Settings → Properties panel

## Files
- `Config.gs` — API keys, constants, helpers
- `License.gs` — license verification logic
- `Menu.gs` — sheet menu items
- `ZeroFormatter.gs` — zero visibility toggle
- `appsscript.json` — GAS manifest
- `.clasp.json` — clasp config
- `site/` — GitHub Pages static site (auto-deployed to `docs/`)

## Branch Strategy
- **master:** Production code
- Feature/hotfix branches: `feature/description`, `hotfix/description`

## Testing
- Deploy via `clasp push`
- Test sheet add-on in Google Sheets
- Verify GitHub Pages site displays correctly

## Dependencies
- **clasp** (installed globally via npm)
- **Gumroad API** (requires product/subscription IDs)
- **Google Sheets API** (via GAS add-on)
- **GitHub Pages** (static hosting from `docs/`)

## Purpose
Google Sheets add-on for hiding zeros in financial data. Integrates with Gumroad for license verification. Includes marketing website on GitHub Pages.
